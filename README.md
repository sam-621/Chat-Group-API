# Chat-Group-API

An API for A web application where you can send messages in a public chat in real time, login, register and see/edit your profile. [Frontend part](https://github.com/rogeliosamuel621/Chat-Group)

Demo: in progress

## Built with

- Node.js - Express
- Socket.io
- Typescript

## Getting started

```
npm install
```

### Development

```
npm run dev
```

### Production

```
npm run start
```
